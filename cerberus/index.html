// Pipeline

What It Does

Ten fully automated phases take a target domain from zero to fully triaged vulnerability report.

PHASE 01 🔍 Reconnaissance

Subdomain enumeration via Subfinder, Amass, and Assetfinder. DNS resolution, live host detection, and scope validation.

PHASE 02 🌐 Web Discovery

HTTP probing with HTTPx. Technology fingerprinting, WAF detection, and response analysis across all discovered hosts.

PHASE 03 🗺️ Content Mapping

Directory and endpoint discovery via Feroxbuster. JavaScript analysis, API endpoint extraction, and sitemap crawling.

PHASE 04 📸 Visual Recon

Gowitness screenshots of all live targets. Visual clustering to identify attack surface patterns and login portals.

PHASE 05 ⚡ Parameter Discovery

Wayback Machine mining, Gau URL extraction, and parameter fuzzing. Hidden parameter identification for injection testing.

PHASE 06 💉 Vulnerability Scanning

Nuclei template scanning across all discovered assets. CVE detection, misconfiguration checks, and known exploit testing.

PHASE 07 🎯 Active Exploitation

XSS, SQLi, SSRF, and IDOR testing with GPU-accelerated fuzzing. Custom wordlists and bypass techniques.

PHASE 08 🤖 AI Triage

Claude Haiku analyzes all findings. Severity scoring, false positive filtering, and exploit chain identification.

PHASE 09 📊 Report Generation

Automated HackerOne-ready reports with CVSS scores, reproduction steps, and remediation recommendations.

PHASE 10 🔄 Continuous Monitoring

Scheduled rescans, diff analysis against previous results, and new asset alerting via n8n automation.

// Installation

Get Started

Requires Windows 11 with WSL2 and Kali Linux. Node.js 18+ required.

terminal
# Clone the repository
$ git clone https://github.com/phantom-suite/cerberus
$ cd cerberus

# Install dependencies
$ npm install

# Run Kali setup script (installs all tools)
$ wsl bash scripts/setup-kali.sh

# Launch CERBERUS
$ npm run dev
