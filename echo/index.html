// Capabilities

Features

The full forensic toolkit for steganography operators.

🔢
LSB Encoding

Least Significant Bit encoding across R, G, B channels. Configurable bit depth — 1 to 4 bits per channel.

✅
Hash Verification

SHA-256 hash embedded alongside payload. Verify data integrity on extraction. Detect tampering automatically.

🔍
Hex Viewer

Full hex dump of carrier images. Navigate byte-by-byte, highlight regions, and annotate suspicious patterns.

🧪
8 Detection Tests

Chi-squared, entropy analysis, histogram anomaly, RS analysis, LSB randomness, and four more statistical tests.

🖼️
Image Comparison

Side-by-side original vs. carrier comparison. Pixel-level diff visualization highlights exactly where data was embedded.

🔐
Pre-Encryption

Encrypt your payload with AES-256 before embedding. The stego layer is just one line of defense.

📊
Histogram Analysis

Color channel histograms with before/after overlay. Visual detection of distribution anomalies caused by encoding.

🎓
CTF Challenges

Graded CTF-style challenges from beginner to expert. Real embedded images. Real extraction. Real flags.
