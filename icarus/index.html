01 // THE ASCENT

From Script to Strike

Four stages of flight. From raw DuckyScript to deployed payload. ICARUS handles the full lifecycle.

01
FORGE
Write DuckyScript in the visual editor with syntax highlighting, auto-complete, and live validation.
02
ENCODE
Compile scripts to inject.bin. Choose keyboard layouts, set execution delays, apply obfuscation.
03
SIMULATE
Test payloads in the deployment simulator. See keystroke timing, window focus, and output preview.
04
DEPLOY
Flash to USB Rubber Ducky or compatible HID device. One-click deploy with status feedback.
02 // ARSENAL

Wings of Wax and Fire

Every tool to craft, test, and deploy USB HID payloads. Engineered for precision.

65+ DUCKYSCRIPT PAYLOADS
Pre-built payloads for recon, exfiltration, persistence, reverse shells, WiFi grabbers, and more.
VISUAL PAYLOAD BUILDER
Drag-and-drop DuckyScript editor with syntax highlighting, line numbers, and real-time validation.
DEPLOYMENT SIMULATOR
Test payloads without hardware. Simulates keystroke injection, timing, and target window behavior.
DUCKYSCRIPT ENCODER
Compile .ds scripts to inject.bin. Supports multiple keyboard layouts and encoding schemes.
PAYLOAD CATEGORIES
Organized into Recon, Exfiltration, Persistence, Privilege Escalation, Network, and Custom categories.
MULTI-PLATFORM TARGETS
Payloads for Windows, macOS, and Linux. Auto-detect target OS or specify manually.
EXFIL WEBHOOKS
Built-in webhook support for data exfiltration. Discord, Slack, HTTP POST, and custom endpoints.
CONFIGURABLE ENGINE
Customize injection speed, default delays, keyboard layout, and output paths. Everything tunable.
