// CAPABILITIES

Weaponized Intelligence

Fifteen core modules working in concert. Seven new systems in v3.0 — achievements, network inspector, NEXUS control, personality engine, session tracking, data export, and performance polish.

REINFORCEMENT LEARNING BRAIN
Q-learning neural network that adapts attack strategies based on environment feedback. Gets smarter with every session, optimizing channel selection and timing.
WARDRIVING MAP
Cyberpunk dark-tiled Leaflet map with real-time GPS plotting. Visualize captured networks, deauth zones, and wardriving paths with heat-map overlays.
PMKID CAPTURE
Clientless PMKID hash extraction for WPA/WPA2 networks. No deauthentication required — passive, fast, and efficient capture with automatic hash formatting.
SPLIT CHIP ARCHITECTURE
CHIMERA GRID handles scanning and recon. NEXUS chip executes attacks via ESP-NOW protocol. Dual-chip design for parallel operations and maximum efficiency.
BOOT SEQUENCE TERMINAL
Cinematic terminal boot animation with system diagnostics, matrix rain, and hardware detection. Sets the mood every time DUSSIGOTCHI wakes up.
SOUND FX + HAPTICS
Web Audio API synthesized sound effects for captures, level-ups, and events. Combined with haptic feedback patterns for a tactile, immersive experience.
LEVEL / XP PROGRESSION
RPG-style experience system. Earn XP from captures, successful deauths, and new network discoveries. 7 ranks from Script Kiddie through Packet God.
CYBERPUNK INTERFACE
OLED-optimized dark UI with matrix rain, adaptive FPS on low battery, neon glows, swipe navigation, and 5 tabs. Every pixel drips with cyberpunk aesthetic.
10 ACHIEVEMENTS
Unlock badges from First Blood to Packet God. Slide-in toast notifications with custom haptic patterns and ascending synth chime on every unlock.
NETWORK INSPECTOR
Tap any network for a bottom-sheet detail view. RSSI sparkline, OUI vendor lookup (150+ manufacturers), attack history, and PMKID/DEAUTH/IGNORE quick actions.
SESSION TRACKING
Start/stop wardrive sessions with live timer, GPS trail, haversine distance calculation, per-session metrics, and exportable session reports.
DATA EXPORT SUITE
WiGLE CSV with proper AuthMode, Hashcat .hc22000, injection-safe CSV, session reports, and native Android share sheet. Every format you need.
